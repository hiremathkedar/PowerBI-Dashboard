# PowerBI-Dashboard

# 📊 Power BI Retail Dashboard – Product Analytics
This project presents a fully interactive **Power BI Dashboard** built using retail transaction data. The dashboard offers clear and actionable insights into product sales, performance by location, customer preferences, and purchasing trends over time.

---

## 📁 Files Included
- `Dashboards.pbit` – Power BI Template file containing all visuals and data model structure.
- `products.csv` – Raw data file used to feed the dashboard.

---

## 🎯 Objectives
- Provide a **comprehensive view of product performance** across various dimensions.
- Help stakeholders and business users make **data-driven decisions** using intuitive visuals.
- Enable **dynamic filtering and drilldowns** based on product, time, location, payment method, and more.

---

## 📌 Dashboard Features
The Power BI dashboard includes:

1. **Sales Overview**  
   - Total Gross & Net Revenue  
   - Discount Impact  
   - Monthly sales trend

2. **Top Performing Products**  
   - Based on Gross and Net Amounts  
   - Dynamic category filtering

3. **Customer Demographics Breakdown**  
   - Age groups and gender  
   - Popular categories by demographic

4. **Geographical Insights**  
   - Purchases by location  
   - Regional sales contribution

5. **Payment Method Analysis**  
   - Usage distribution across digital, card, and net banking

6. **Time-based Analysis**  
   - Peak hours and days  
   - Seasonal patterns by month

## ▶️ How to Use
1. Open `Dashboards.pbit` in Power BI Desktop.
2. When prompted, **load `products.csv`** as the data source.
3. Explore the dashboard interactively:
   - Click on any chart or slicer to filter others dynamically.
   - Hover over visuals for tooltip details.
   - Export filtered visuals if needed.
